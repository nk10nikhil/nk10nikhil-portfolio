# Nikhil Kumar's Portfolio

A modern, responsive portfolio web application built with **React**, **React Router**, and **Tailwind CSS**. This project showcases my skills, projects, and contact information, and includes several interactive mini-apps.

## 🚀 Features

- **Home, About, Skills, Projects, Contact** pages
- Modern UI with Tailwind CSS
- React Router for seamless navigation
- Interactive mini-projects:
  - Background Color Changer
  - Password Generator
  - Currency Converter
  - Todo App
  - Theme Switcher
- GitHub integration (fetches profile data)
- Responsive and mobile-friendly design

## 📦 Getting Started

1. **Clone the repository:**

   ```sh
   git clone https://github.com/nk10nikhil/your-repo.git
   cd your-repo
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Run the development server:**

   ```sh
   npm run dev
   ```

4. **Build for production:**
   ```sh
   npm run build
   ```
